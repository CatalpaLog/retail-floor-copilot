# 门店运营操作手册

## 导购

1. 扫描吊牌或输入商品编码。
2. 查看商品话术、尺码和库存。
3. 遇到退款、赔偿、活动叠加或客诉问题，提交店长确认。
4. 在通知中心查看审批结果。
5. 回答有误时提交反馈。
6. 扫描未知商品时提交新品建档申请。

## 店长

1. 关注侧边栏高风险和中风险数量。
2. 优先处理超时工单。
3. 核验顾客、商品、凭证和门店权限。
4. 确认、修改、转线下或升级区域运营。
5. 处理本门店Bad Case，必要时提交知识优化申请。
6. 补充新品资料并提交区域审核。
7. 使用门店看板识别高频问题、风险导购和知识缺口。

## 区域运营

1. 处理店长升级工单和跨门店争议。
2. 聚合同类Bad Case，分配责任人和门店任务。
3. 发布规则并跟踪咨询量变化。
4. 维护商品卡、禁语、尺码和连带推荐。
5. 审核新品编码、版本、价格和有效期。
6. 查看区域门店对比、活跃度、零结果率和高风险分布。
7. 定期导出Excel或看板图片。
8. 通过操作日志追溯规则、商品和审批变更。

## 旺场操作

1. 开启“旺场简洁模式”。
2. 使用语音或文字提出问题。
3. 第一屏直接读取顾客沟通话术。
4. 出现禁语或“需店长确认”时停止承诺。
5. 商品需要算价时点击“加入快捷算价”。

## 店长日常闭环

1. 登录后先查看高风险与中风险待办数量。
2. 对等待超过2小时的工单优先处理。
3. 信息不全时标记“处理中”并核验订单、商品和规则。
4. 超权限问题填写原因并升级区域运营。
5. 每日查看Bad Case，分配责任人和处理期限。
6. 对知识库问题提交区域优化申请。

## 区域运营日常闭环

1. 优先处理门店升级工单。
2. 按门店和风险筛选区域问题。
3. 对同类问题批量分派或统一修改规则/商品口径。
4. 发布规则后检查门店已读情况。
5. 通过看板查看知识覆盖、高风险、活跃度与门店综合得分。
6. 每周导出报表，复盘规则、商品和培训动作效果。
